## Security Use Cases Covered

### Security Operations Detection & Investigation
- Brute force and password spray attacks  
- Suspicious PowerShell and command-line activity  
- Malware alert investigation and validation  
- Abnormal authentication behavior  
- Privileged account activity monitoring  

### Correlated & Context-Aware Detections
- Lateral movement indicators (RDP / SMB activity)  
- Privilege escalation attempts (on-prem and Azure AD)  
- Credential misuse and anomalous account behavior  
- Suspicious service and scheduled task creation

- Cloud resource abuse and anomalous Azure activity  
- Living-off-the-Land (LOLBins) activity detection  
- Insider threat behavioral indicators  
- Persistent and low-and-slow attack patterns
